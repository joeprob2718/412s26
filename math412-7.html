Heat Equation on the Real Line

Instead of a rod occupying the finite domain $0 \le x \le L$, we consider

$$\begin{cases} \text{PDE:} & \dfrac{\partial u}{\partial t} = k\,\dfrac{\partial^2 u}{\partial x^2}, \quad -\infty < x < \infty,\; t > 0 \\[10pt] \text{IC:} & u(x,0) = f(x), \quad -\infty < x < \infty \end{cases}$$
The spatial domain is now the entire real line $(-\infty, \infty)$ — no finite boundary!

For physical reasons (so the total thermal energy has a chance to be bounded), we consider initial conditions $f(x)$ satisfying

$$\lim_{x \to \pm\infty} f(x) = 0,$$

and seek solutions that satisfy

$$\lim_{x \to \pm\infty} u(x,t) = 0 \quad \text{for all } t > 0.$$

2/10

Therefore, we consider

$$\begin{cases} \text{PDE:} & \dfrac{\partial u}{\partial t} = k\,\dfrac{\partial^2 u}{\partial x^2}, \quad -\infty < x < \infty,\; t > 0 \\[10pt] \text{"BC":} & u(-\infty, t) = u(\infty, t) = 0 \quad t > 0 \\[10pt] \text{IC:} & u(x,0) = f(x), \quad -\infty < x < \infty \end{cases}$$

where $u(\pm\infty, t)$ are understood as limits.

3/10

As usual, let's first seek product solutions of the form

$$u(x,t) = \phi(x)\,G(t),$$

leading to

$$\frac{G'(t)}{k\,G(t)} = \frac{\phi''(x)}{\phi(x)} =: -\lambda,$$

yielding the coupled ODEs

$$\begin{cases} \phi''(x) = -\lambda\,\phi(x) \\[4pt] G'(t) = -\lambda\, k\, G(t) \end{cases}$$

4/10

Boundary Conditions

You might guess that we want to enforce $\phi(\pm\infty) = 0$ to guarantee $u(\pm\infty, t) = 0$ for all $t > 0$.

However, that won't allow for any nontrivial solutions for $\phi(x)$, and so we instead require that $\phi(x)$ is bounded for all $x$.

Once we superimpose solutions, we'll be able to see that this condition is sufficient to guarantee $u(\pm\infty, t) = 0$.

The book writes this as $|\phi(\pm\infty)| < \infty$, although I'm not a fan of this notation since it suggests the limits exist — this is the case when writing $u(\pm\infty, t) = 0$. Perhaps a better way to write this would be: there exists a number $C > 0$ so that

$$|\phi(x)| < C \quad \text{for all } -\infty < x < \infty,$$

or just say $\phi(x)$ is bounded on the entire real line.

1 The distinction wasn't important when considering the region exterior to a disk when we solved the Laplacian in polar coordinates — requiring boundedness ended up requiring the limit existed and was zero. As we'll see, we get cosines and sines here whose limits don't exist as $x \to \pm\infty$.

5/10

Eigenvalue Analysis

Therefore, our coupled ODEs and boundary conditions are

$$\begin{cases} \phi''(x) = -\lambda\,\phi(x), & -\infty < x < \infty, \quad \phi(x) \text{ is bounded} \\[6pt] G'(t) = -\lambda\, k\, G, & t \ge 0 \end{cases}$$
Characteristic polynomial: $r^2 + \lambda = 0$.

As the spatial eigenvalue problem has "boundary conditions" (boundedness), this is our eigenvalue problem that we'll use to find the possible values of the separation constant $\lambda$.

Case $\lambda < 0$

The solution is

$$\phi(x) = c_1\cosh(\sqrt{-\lambda}\,x) + c_2\sinh(\sqrt{-\lambda}\,x),$$

(or a linear combination of $e^{\pm\sqrt{-\lambda}\,x}$).

$\cosh x = \frac{e^x + e^{-x}}{2}$, $\sinh x = \frac{e^x - e^{-x}}{2}$

However, both $|\cosh(x)|$ and $|\sinh(x)|$ diverge to $\infty$ at both $x \to \pm\infty$, and so boundedness would require $c_1 = c_2 = 0$. (Similarly, $e^{\sqrt{-\lambda}\,x}$ is unbounded as $x \to +\infty$, and $e^{-\sqrt{-\lambda}\,x}$ is unbounded as $x \to -\infty$, hence only the trivial solution is bounded for all $x$.)

Therefore, nontrivial solutions cannot exist for $\lambda < 0$.

6/10

Cases $\lambda = 0$ and $\lambda > 0$

For $\lambda = 0$ we have bounded constant solutions

$$\phi(x) = c_1$$

for any $c_1$.

If $\lambda > 0$ we have bounded solutions

$$\phi(x) = c_1\cos(\sqrt{\lambda}\,x) + c_2\sin(\sqrt{\lambda}\,x)$$

for any constants $c_1, c_2$.

For Fourier series problems on bounded domains, the allowed eigenvalues were discrete. However, this spectrum (set for which the differential operator $\left(\dfrac{d^2}{dx^2} + \lambda\right)$ is not invertible) is a continuous spectrum — any $\lambda \ge 0$ is allowed.2

2 Technically the word "eigenvalue" is reserved for elements of the spectrum possessing associated square-integrable eigenfunctions — these functions here would be called generalized eigenfunctions since they are not square integrable. However, superimposing them appropriately will yield square-integrable functions $u(x,t)$.

7/10

Product Solutions & Continuous Superposition

If $\lambda = 0$ then $G(t) = C$, and if $\lambda > 0$ it follows $G(t) = Ce^{-\lambda k t}$.

Putting it together, possible product solutions are

$$u_p(x,t) = \cos(\sqrt{\lambda}\,x)\,e^{-\lambda k t} \quad \text{or} \quad \sin(\sqrt{\lambda}\,x)\,e^{-\lambda k t}, \quad \lambda \ge 0.$$

Instead of a summation of the (countable) discrete values of $\lambda$, we can write a typical solution as an integral ("continuous sum"):

$$u(x,t) = \int_0^{\infty}\left[c_1(\lambda)\cos(\sqrt{\lambda}\,x)\,e^{-\lambda k t} + c_2(\lambda)\sin(\sqrt{\lambda}\,x)\,e^{-\lambda k t}\right]d\lambda,$$

where $c_1(\lambda), c_2(\lambda)$ are arbitrary functions of the continuous variable $\lambda$ (which is why we're not using subscript sequence notation).

Under suitable hypotheses on $c_1(\cdot), c_2(\cdot)$, we can justify differentiating under the integral sign to verify this "continuous sum" satisfies the PDE, which we can call the generalized superposition principle.

8/10

Connection to the Fourier Transform

Since $\lambda \ge 0$, let's write $\lambda = \omega^2$ for a real frequency $\omega \ge 0$, so that we have3

$$u(x,t) = \int_0^{\infty}\left[A(\omega)\cos(\omega x)\,e^{-k\omega^2 t} + B(\omega)\sin(\omega x)\,e^{-k\omega^2 t}\right]d\omega.$$

Compare this to our solution on the bounded region $[-L, L]$ with periodic boundary conditions: the allowable frequencies were discrete,

$\omega_n = \dfrac{n\pi}{L}$

so we had an infinite sum:

$$u(x,t) = \sum_{n=0}^{\infty} a_n\cos\frac{n\pi x}{L}\,e^{-k(n\pi/L)^2 t} + \sum_{n=1}^{\infty} b_n\sin\frac{n\pi x}{L}\,e^{-k(n\pi/L)^2 t}.$$

3 Since $d\lambda = 2\omega\,d\omega$, these new arbitrary functions satisfy $A(\omega) = 2\sqrt{\lambda}\,c_1(\lambda)$, similar for $B(\omega)$. However, you're used to relabeling arbitrary constants in ODE general solutions from Math 250/1, this is the same idea but for arbitrary functions!

9/10

Warning: Many texts reserve $\omega$ for representing a temporal frequency and use $k$ for spatial frequencies. We're used to using $k$ for thermal diffusivity, so we'll use $\omega$ for a spatial frequency here.

Then, implementing the initial condition $u(x,0) = f(x)$, we need

$$f(x) = u(x,0) = \int_0^{\infty}\left[A(\omega)\cos(\omega x) + B(\omega)\sin(\omega x)\right]d\omega.$$

Similar to how $(a_n), (b_n)$ were the Fourier coefficients for $f(x)$ in the $[-L, L]$ periodic BC problem; these functions $A(\omega), B(\omega)$ will be related to what we'll call the Fourier transform of $f(x)$.

1. Laplace's Equation in 2D

Slides 2

The problems we set up and solved for the 1D heat equation (zero-temperature ends, insulated ends, and a ring/periodic boundary conditions) were initial-boundary value problems (IBVP) on "half-infinite rectangles", e.g.

\[ \begin{cases} \text{PDE:} & \pd{u}{t} = k\,\pdd{u}{x}, \quad 0 < x < L,\; t > 0 \\[6pt] \text{BC:} & u(0,t) = u(L,t) = 0, \quad t > 0 \\[4pt] \text{IC:} & u(x,0) = f(x), \quad 0 \le x \le L \end{cases} \]
IBVP — solve for \(u(x,t)\) for \(0 < x < L\) and all \(t > 0\). The boundary conditions \(u(0,t)\) and \(u(L,t)\) are prescribed for \(t > 0\); the initial condition \(u(x,0) = f(x)\) is prescribed for \(0 \le x \le L\).

2. 2D Laplace in a Rectangle — Prescribed Boundary Values

Slide 3

For well-posedness related reasons (discussed later), we are not interested in solving the 2D Laplace equation on a half-infinite strip. Instead, we consider a "pure" BVP of the form:

\(\nabla^2 u = 0\) — solve for \(u(x,y)\).
\[ \begin{cases} \text{PDE:} & \pdd{u}{x} + \pdd{u}{y} = 0, \quad 0 < x < L,\; 0 < y < H \\[6pt] \text{BC1:} & u(0,y) = g_1(y), \quad 0 \le y \le H \\[4pt] \text{BC2:} & u(L,y) = g_2(y), \quad 0 \le y \le H \\[4pt] \text{BC3:} & u(x,0) = f_1(x), \quad 0 \le x \le L \\[4pt] \text{BC4:} & u(x,H) = f_2(x), \quad 0 \le x \le L \end{cases} \]
The rectangle has boundary data prescribed on all four sides: \(u = g_1\) on the left, \(u = g_2\) on the right, \(u = f_1\) on the bottom, \(u = f_2\) on the top. The rectangle has width \(L\) and height \(H\).

Footnote: If we wanted to think of \(y\) as "time", this would be an "initial-final" value problem — i.e., a boundary value problem.

3. Superposition of Solutions

Slide 4

The PDE is linear homogeneous, but the BC are not homogeneous.

Intuitively, the idea of superposition we used for the 1D heat equation can handle "one inhomogeneity" at a time, not four. We can superimpose superimposed solutions!

\(\nabla^2 u_1 = \nabla^2 u_2 = \nabla^2 u_3 = \nabla^2 u_4 \equiv 0 \implies \nabla^2 u = 0\).

Figure 2.5.1 — Laplace's equation inside a rectangle (Superposition Decomposition)

\(u = u_1 + u_2 + u_3 + u_4\)

Each \(u_k\) solves \(\nabla^2 u_k = 0\) with one inhomogeneous boundary side and zeros on the other three:

\(u_1\): bottom = \(f_1(x)\), others = 0  |  \(u_2\): right = \(g_2(y)\), others = 0
\(u_3\): top = \(f_2(x)\), others = 0  |  \(u_4\): left = \(g_1(y)\), others = 0

That is,

\[ u(x,y) = u_1(x,y) + u_2(x,y) + u_3(x,y) + u_4(x,y) \]

solves the BVP with all four inhomogeneous boundary conditions.

4. Solving with One Inhomogeneous Side (\(u_4\))

Slides 5–6
\[ \begin{cases} \text{PDE:} & \pdd{u_4}{x} + \pdd{u_4}{y} = 0, \quad 0 < x < L,\; 0 < y < H \\[6pt] \text{BC1:} & u_4(0,y) = g_1(y), \quad 0 \le y \le H \\[4pt] \text{BC2:} & u_4(L,y) = 0, \quad 0 \le y \le H \\[4pt] \text{BC3:} & u_4(x,0) = 0, \quad 0 \le x \le L \\[4pt] \text{BC4:} & u_4(x,H) = 0, \quad 0 \le x \le L \end{cases} \]

Separation of variables

Set up a product solution ansatz:

\[ u_p(x,y) = h(x)\,\phi(y) \implies \frac{h''(x)}{h(x)} + \frac{\phi''(y)}{\phi(y)} = 0 \]
Note: \(\frac{\partial^2 u}{\partial x^2} = h''\phi\) and \(\frac{\partial^2 u}{\partial y^2} = h\phi''\). Then \(h''\phi + h\phi'' = 0\); dividing by \(h\phi\) gives the separated equation.

Since one side depends only on \(x\) and the other only on \(y\), both must equal a constant:

\[ -\frac{h''(x)}{h(x)} = \frac{\phi''(y)}{\phi(y)} =: -\lambda \]

Ignoring the inhomogeneous BC for now (will be achieved via superposition later), we obtain the coupled ODE system:

\[ \begin{cases} \phi''(y) = -\lambda\,\phi(y), & \phi(0) = \phi(H) = 0 \\[6pt] h''(x) = \lambda\,h(x), & h(L) = 0 \end{cases} \]
The \(\phi\) equation with homogeneous Dirichlet BCs is the same eigenvalue problem from the heat equation.

Recycling previous analysis for the heat equation, nontrivial \(\phi\) are given by

\[ \phi(y) = B\sin\!\left(\frac{n\pi y}{H}\right), \quad n = 1, 2, 3, \ldots \]

corresponding to \(0 < \lambda = \left(\frac{n\pi}{H}\right)^2\).

Since \(\lambda > 0\), the equation \(h'' = \lambda h\) has characteristic polynomial \(r^2 - \left(\frac{n\pi}{H}\right)^2 = 0\), giving \(r = \pm\frac{n\pi}{H}\). So the characteristic roots are real.

This guarantees real characteristic roots for the \(x\)-ODE:

\[ h(x) = c_1\,e^{n\pi x/H} + c_2\,e^{-n\pi x/H}, \qquad h(L) = 0. \]

5. Hyperbolic Trigonometric Functions

Slides 7–9
Definition (Hyperbolic Trig). \[ \sinh(x) := \frac{e^x - e^{-x}}{2}, \qquad \cosh(x) := \frac{e^x + e^{-x}}{2}. \]

Key properties:

\(\sinh(x)\) is odd, \(\cosh(x)\) is even.

\(\sinh(0) = 0\),   \(\cosh(0) = 1\).

\(\frac{d}{dx}\sinh(x) = \cosh(x)\).

\(\frac{d}{dx}\cosh(x) = \sinh(x)\).

\(\cosh^2(x) - \sinh^2(x) = 1\).

Connection via Euler's formula:

\[ \cosh(ix) = \cos x, \qquad \sinh(ix) = i\sin(x). \]

Why use hyperbolic trig?

In the same way that

\[ h''(x) = \underbrace{\lambda}_{<0}\,h(x) \implies h(x) = c_1\cos(\sqrt{-\lambda}\,x) + c_2\sin(\sqrt{-\lambda}\,x), \]

we have

\[ h''(x) = \underbrace{\lambda}_{>0}\,h(x) \implies h(x) = c_1\cosh(\sqrt{\lambda}\,x) + c_2\sinh(\sqrt{\lambda}\,x). \]

We've changed the basis for the 2D solution space from \(\{e^{\sqrt{\lambda}\,x},\, e^{-\sqrt{\lambda}\,x}\}\) to \(\{\cosh(\sqrt{\lambda}\,x),\, \sinh(\sqrt{\lambda}\,x)\}\). This is more convenient for analyzing boundary values since \(\cosh(0) = 1\) and \(\sinh(0) = 0\).

6. Shifted Basis & Applying the BC

Slides 10–11

We can also shift the basis functions:

\[ h''(x) = \underbrace{\lambda}_{>0}\,h(x) \implies h(x) = c_1\cosh\!\bigl[\sqrt{\lambda}\,(x - x_0)\bigr] + c_2\sinh\!\bigl[\sqrt{\lambda}\,(x - x_0)\bigr] \]

for any \(x_0\). (Justified by linear ODE theory — any \(n\) linearly independent solutions span the full solution space; linear independence can be checked via the Wronskian.)

Therefore, instead of imposing a compatibility condition on \(c_1, c_2\) to satisfy \(h(L) = 0\) using exponentials, we choose \(x_0 = L\):

\[ h(x) = c_1\cosh\!\left(\frac{n\pi(x-L)}{H}\right) + c_2\sinh\!\left(\frac{n\pi(x-L)}{H}\right). \]
At \(x = L\): \(h(L) = c_1\cosh(0) + c_2\sinh(0) = c_1 \cdot 1 + c_2 \cdot 0 = c_1\). So \(h(L) = 0 \implies c_1 = 0\).

Then

\[ h(L) = 0 \implies c_1 = 0 \implies h(x) = c\,\sinh\!\left(\frac{n\pi(x-L)}{H}\right) \]

is a nontrivial solution for any \(c\), since \(\sinh(0) = 0\). (This was the whole point of using hyperbolic trig!)

7. Product Solutions & Coefficient Formulas

Slides 12–16

Nontrivial product solutions to the BVP (with three homogeneous BCs) are:

\[ u_p(x,y) = h(x)\,\phi(y) = B\sinh\!\left(\frac{n\pi(x-L)}{H}\right)\sin\!\left(\frac{n\pi y}{H}\right), \quad n = 1,2,\ldots \]

By superposition, the general solution is:

\[ u_4(x,y) = \sum_{n=1}^{\infty} B_n \sinh\!\left(\frac{n\pi(x-L)}{H}\right)\sin\!\left(\frac{n\pi y}{H}\right). \]

Applying the inhomogeneous BC

Recall BC1: \(u_4(0,y) = g_1(y)\). Setting \(x = 0\):

\[ g_1(y) = -\sum_{n=1}^{\infty} B_n\sinh\!\left(\frac{n\pi L}{H}\right)\sin\!\left(\frac{n\pi y}{H}\right) \]
Note: \(\sinh\!\bigl(\frac{n\pi(0 - L)}{H}\bigr) = \sinh\!\bigl(\frac{-n\pi L}{H}\bigr) = -\sinh\!\bigl(\frac{n\pi L}{H}\bigr)\) since \(\sinh\) is odd.

Multiply both sides by \(\sin\!\left(\frac{m\pi y}{H}\right)\), integrate from \(0\) to \(H\), and use orthogonality:

Orthogonality relation (integers \(n,m \ge 1\)): \(\displaystyle\int_0^H \sin\!\left(\frac{n\pi y}{H}\right)\sin\!\left(\frac{m\pi y}{H}\right)dy = \begin{cases} 0 & n \ne m \\ \frac{H}{2} & n = m\end{cases}\)

Only the \(n = m\) term survives on the right, giving:

\[ \int_0^H g_1(y)\sin\!\left(\frac{m\pi y}{H}\right)dy = -B_m\sinh\!\left(\frac{m\pi L}{H}\right)\frac{H}{2} \]

Solving for \(B_m\):

\[ \boxed{\;B_m = \frac{-2}{H\sinh\!\left(\frac{m\pi L}{H}\right)}\int_0^H g_1(y)\sin\!\left(\frac{m\pi y}{H}\right)dy.\;} \]

Alternative derivation using the reference table

The Fourier sine series formula (renaming \(x \to y\) and \(L \to H\)) states:

\[ f(y) = \sum_{n=1}^{\infty}\underbrace{B_n}_{\text{coeff}}\sin\!\left(\frac{n\pi y}{H}\right) \implies \underbrace{B_n}_{\text{coeff}} = \frac{2}{H}\int_0^H f(y)\sin\!\left(\frac{n\pi y}{H}\right)dy. \]

Identifying \(-B_n\sinh\!\left(\frac{n\pi L}{H}\right)\) as the Fourier coefficient of \(g_1(y)\) immediately yields the same result.

Full solution

This completes the solution for \(u_4(x,y)\). The same idea is applied for \(u_1, u_2, u_3\), so that:

\[ u(x,y) = u_1(x,y) + u_2(x,y) + u_3(x,y) + u_4(x,y) \]

solves the full BVP with all four inhomogeneous boundary conditions.

8. Well-Posedness

Slides 15–16 (original numbering)
Definition (Well-Posed Problem). A problem is well-posed when:
  1. There exists a solution. (existence of solutions)
  2. The solution is unique. (uniqueness of solutions)
  3. The solution depends continuously on the inhomogeneous data. (continuous dependence on data)

The first two items indicate "we understand the natural laws governing this scenario," and the third means our understanding is stable with respect to unavoidable measurement errors.

PDEs that break condition #3 exhibit discontinuous dependence on data — small perturbations in the data lead to huge changes in the solution.
Chaos ≠ Ill-Posed! Chaos involves: (1) aperiodic long-term behavior, (2) in a deterministic system, (3) exhibiting sensitive (but continuous) dependence on initial data, so predictions at \(t = O(1)\) are useless.

Examples: the double pendulum and the Lorenz system (3 ODEs modeling convection rolls). Turbulent fluid flow (Navier–Stokes with boundary effects) seems to be PDE chaos, but well-posedness for nonlinear fluid PDEs is a million-dollar question (Clay Millennium Problem).

9. IBVP for Laplace Equation Is Ill-Posed

Slides 17–18

Consider the following IBVP on the half-strip \((0,\pi)\times(0,\infty)\) with \(L = \pi\):

\[ \begin{cases} \pdd{u_n}{t} + \pdd{u_n}{x} = 0, & 0 < x < L,\; t > 0 \\[6pt] u_n(0,t) = u_n(\pi,t) = 0, & t > 0 \\[4pt] u_n(x,0) = 0, & 0 \le x \le L \\[4pt] \pd{u_n}{t}(x,0) = f_n(x) := \frac{1}{n}\sin(nx), & 0 \le x \le L \end{cases} \]

(Unique) solutions are:

\[ u_n(x,t) = \frac{1}{n^2}\sin(nx)\sinh(nt). \]
As \(n \to \infty\): the data \(f_n(x) = \frac{1}{n}\sin(nx) \to 0\) for all \(x\) (oscillates between \(\pm 1/n\)).

The unique solution to the problem with zero data \(f \equiv 0\) is \(u(x,t) \equiv 0\).

However, as \(n \to \infty\), \(u_n\) takes on arbitrarily large positive and negative values for all positive times (for any \(T >0, \lim_{n \to \infty} \frac{\sinh(nt)}{n^2} \rightarrow + \infty\) since \(\sinh(\cdot)\) involves exponential growth). Therefore, the solutions do not depend continuously on the data \(f\) near "\((f,u) \equiv (0,0)\)."

The data \(f_n \to 0\), but the corresponding solutions \(u_n(x,t) \not\to u(x,t) = 0\). In fact, \(u_n\) blows up! This is the hallmark of ill-posedness.

10. Classification of Linear PDE Types

Slides 21–23

▸ Parabolic — prototype: heat equation

\[\pd{u}{t} = k\nabla^2 u\]

IBVPs are well-posed forward in time. Infinite speed of propagation. Space-time mean value properties. Maximum principles.

Symbol: \(P(\tau,\xi) = \tau - k\xi^2\); level sets are parabolas.

▸ Elliptic — prototype: Laplace's equation

\[\nabla^2 u = 0\]

Boundary value problems are well-posed; initial value problems are not. Mean value property. Maximum principles.

Symbol: \(P(\xi,\eta) = \xi^2 + \eta^2\); level sets are ellipses (circles).

▸ Hyperbolic — prototype: wave equation

\[\pdd{u}{t} = c^2\nabla^2 u\]

IBVPs are well-posed forward and backward in time. Finite speed of propagation.

Symbol: \(P(\tau,\xi) = \tau^2 - c^2\xi^2\); level sets are hyperbolas.

The nomenclature comes from treating the PDE operators as polynomials and examining the level sets of the resulting "symbol." Rearrange the PDE to be of the form \( Pu = 0\), where \(P\) is a polynomial differential operator. Then, in the above replace \( \frac{\partial}{\partial t} \to \tau, \frac{\partial}{\partial x} \to \xi, \frac{\partial}{\partial y}\to \eta.\)

Nonlinear equations: There isn't a single precise definition to classify type in every context; often a nonlinear equation is classified (e.g., as hyperbolic) by establishing some of the qualitative properties that type tends to have.

11. Backwards Heat Equation Is Ill-Posed

Slides 24–25
Separation of variables: \(u = \phi(x)G(t)\) gives \(\phi'' = -\lambda\phi\), \(\phi(0)=\phi(\pi)=0\) and \(G' = k\lambda G\) (note positive sign — growth!). Compare to the forward heat equation where \(G' = -k\lambda G\) (decay).

Consider the backward heat equation (\(L = \pi\)):

\[ \begin{cases} \pd{u_n}{t} = -k\,\pdd{u_n}{x}, & 0 < x < L,\; t > 0 \\[6pt] u_n(0,t) = u_n(\pi,t) = 0, & t > 0 \\[4pt] u_n(x,0) = \frac{1}{n}\sin(nx), & 0 \le x \le L \end{cases} \]

(Unique) solutions:

\[ u_n(x,t) = \frac{1}{n}\sin(nx)\,e^{kn^2 t}. \]
The initial data is just one eigenfunction \(\frac{1}{n}\sin(nx)\), which goes to zero pointwise. As \(n \to \infty \) solution \(u_n(x,t)\) takes on arbitrarily large positive and negative values for any \(T>0\): fix any \(T > 0\): \(\lim_{n\to\infty}\frac{e^{kn^2 T}}{n} \to \infty\).

As \(n \to \infty\), \(f_n(x) \to 0\) for all \(x\), but the solutions blow up for any \(t > 0\). Same argument as before: the unique solution for zero data is identically zero, but \(u_n\) does not converge to it. The backward heat equation is ill-posed.

This is "more difficult" than the forward heat equation: forward heat equation has exponential decay \(e^{-kn^2t}\), backward has exponential growth \(e^{kn^2t}\).

12. Laplacian in Polar Coordinates

Slides 28–29
Coordinate transformation: \((x,y) \to (r,\theta)\) where \(x = r\cos\theta\), \(y = r\sin\theta\), and \(r = \sqrt{x^2+y^2}\), \(\tan\theta = y/x\).

Using the multivariable chain rule:

\[ \pd{}{x} = \cos\theta\,\pd{}{r} - \frac{\sin\theta}{r}\,\pd{}{\theta} \]
\[ \pd{}{y} = \sin\theta\,\pd{}{r} + \frac{\cos\theta}{r}\,\pd{}{\theta} \]

Squaring each and adding (after careful computation where cross terms cancel):

\[ \nabla^2 = \pdd{}{r} + \frac{1}{r}\pd{}{r} + \frac{1}{r^2}\pdd{}{\theta} = \frac{1}{r}\pd{}{r}\!\left(r\pd{}{r}\right) + \frac{1}{r^2}\pdd{}{\theta} \]
Expanding \(\frac{1}{r}\frac{\partial}{\partial r}\!\left(r\frac{\partial u}{\partial r}\right) = \frac{1}{r}\!\left(1\cdot\frac{\partial u}{\partial r} + r\,\frac{\partial^2 u}{\partial r^2}\right) = \frac{1}{r}\frac{\partial u}{\partial r} + \frac{\partial^2 u}{\partial r^2}\).

13. Laplace's Equation in a Disk

Slides 29–35
\[ \begin{cases} \nabla^2 u = \dfrac{1}{r}\pd{}{r}\!\left(r\pd{u}{r}\right) + \dfrac{1}{r^2}\pdd{u}{\theta} = 0, & 0 < r < a,\; -\pi < \theta \le \pi \\[10pt] u(a,\theta) = f(\theta), & -\pi \le \theta \le \pi \end{cases} \]
Interior of a disk of radius \(a\). The boundary condition is prescribed on the circle \(r = a\).

Coordinate boundary conditions

The \(r = a\) boundary is physical. We also need conditions at the "coordinate boundaries" \(r = 0\) and \(\theta = \pm\pi\):

The origin doesn't have a well-defined angle (up to \(\pm 2\pi\))! Polar coordinates are singular/degenerate at \(r = 0\) — the Laplacian is not defined there.

Boundedness at the origin:

\[|u(0,\theta)| < \infty.\]

Periodicity ("perfect thermal contact"):

\[ u(r,-\pi) = u(r,\pi), \qquad \pd{u}{\theta}(r,-\pi) = \pd{u}{\theta}(r,\pi). \]

Separation of variables

Product solutions \(u_p = \phi(\theta)\,G(r)\) satisfy:

\[ \begin{cases} \phi''(\theta) = -\lambda\,\phi(\theta), & \phi(-\pi) = \phi(\pi),\; \phi'(-\pi) = \phi'(\pi) \\[6pt] r^2G''(r) + rG'(r) = \lambda\,G(r), & |G(0)| < \infty \end{cases} \]
The \(\phi\) BVP is the same periodic eigenvalue problem from the 1D circular ring! (3rd column on reference sheet, with \(L = \pi\).)

Using previous analysis:

\[ \phi(\theta) = \sin(n\theta) \;\text{ or }\; \cos(n\theta), \quad \text{corresponding to } \lambda = n^2, \quad n = 0,1,2,\ldots \]

The radial ODE — Cauchy–Euler equation

\[r^2 G''(r) + rG'(r) - n^2 G(r) = 0.\]

This is a variable-coefficient ODE, so the usual characteristic polynomial technique (constant coefficients only) doesn't work. However, it is an equidimensional (Cauchy–Euler) equation.

Ansatz \(G(r) = r^p\). Then \(G' = pr^{p-1}\), \(G'' = p(p-1)r^{p-2}\). Substituting: \(r^p\bigl[p(p-1) + p - n^2\bigr] = 0\), so \(p^2 - n^2 = 0 \implies p = \pm n\).

For \(n \ge 1\): \(G(r) = c_1 r^n + c_2 r^{-n}\).

For \(n = 0\): \(G(r) = c_1 + c_2 \ln r\).

Boundedness at origin \(|G(0)| < \infty\) requires \(c_2 = 0\) in all cases (both \(r^{-n}\) and \(\ln r\) blow up as \(r \to 0\)).

Full series solution

\[ \boxed{\;u(r,\theta) = \sum_{n=0}^{\infty} A_n r^n \cos(n\theta) + \sum_{n=1}^{\infty} B_n r^n \sin(n\theta)\;} \]

Applying the boundary condition \(u(a,\theta) = f(\theta)\):

\[ f(\theta) = \sum_{n=0}^{\infty} A_n a^n \cos(n\theta) + \sum_{n=1}^{\infty} B_n a^n \sin(n\theta) \]

Standard orthogonality gives the coefficients:

\[ A_0 = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(\theta)\,d\theta \] \[ A_n a^n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(\theta)\cos(n\theta)\,d\theta \] \[ B_n a^n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(\theta)\sin(n\theta)\,d\theta \]
On the Cauchy–Euler ODE: This is basically the only higher-order variable-coefficient ODE for which "nice" closed-form solutions exist. Typically, power series or Frobenius series are the best we can do (discussed in Math 411). The logarithmic solutions (\(\ln r\), \(r^{-n}\)) are often irrelevant for applications since they're unbounded toward \(r \to 0\) or \(r \to \infty\), and physical boundary conditions enforce boundedness. The second logarithmic solution can be obtained from scratch via reduction of order.

14. Mean Value Property

Slides 35–36

At the origin we obtained:

\[ u(0,\theta) = A_0 = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(\theta)\,d\theta \]
This is the average value of \(f\) on the circle \(r = a\)! Recall: average of \(f(x)\) on \([a,b]\) is \(\frac{1}{b-a}\int_a^b f(x)\,dx\).
Mean Value Property. Suppose \(u\) solves Laplace's equation in a region \(R\) in the plane. Then for any point \(P \in R\) and any radius \(r_0\) such that the circle of radius \(r_0\) centered at \(P\) lies within \(R\):

The temperature at any point is the average of the temperature along any circle (lying in \(R\)) centered at that point. \[ u(x_0,y_0) = \frac{1}{2\pi r_0^2}\iint_{\text{disk}} u(r,\theta)\,dA \]
This makes sense with physical intuition about steady-state temperature distributions!

15. Maximum Principle

Slides 36–39
Maximum Principle (no sources). In steady state, the temperature cannot attain its maximum (or minimum) in the interior unless the temperature is constant everywhere.
Laplace solutions can't have strict local mins or local max's in the interior!

Argument via the second derivative test

In Math 230, a nondegenerate local maximum (classified via the second derivative test) satisfies:

\[ D := \pdd{u}{x}\cdot\pdd{u}{y} - \left(\frac{\partial^2 u}{\partial x\,\partial y}\right)^2 > 0, \qquad \pdd{u}{x} < 0. \]
"Hessian determinant" — need both second partials nonzero and same sign!

But for solutions to Laplace's equation:

\[ \nabla^2 u = \pdd{u}{x} + \pdd{u}{y} = 0 \implies \pdd{u}{x} = -\pdd{u}{y} \]

So the two second partial derivatives have opposite signs — the conditions for a nondegenerate local max (or min) can never be satisfied.

Similarly, a nondegenerate local minimum requires \(D > 0\) and \(\frac{\partial^2 u}{\partial x^2} > 0\), which also fails.

What about degenerate critical points?

Degenerate critical points (\(D = 0\)) are possible — for example, monkey saddles like \(f(x,y) = x^4 + y^4\) (strict local min at \((0,0)\) but \(D = 0\) — not classifiable by the 2nd derivative test).

To handle this case: suppose \(u\) has a strict local maximum. By the mean value property, the maximum value must equal the average of all points on any surrounding circle. But if the value is strictly greater than all nearby values, it cannot equal their average — contradiction!

Physical intuition: \(\nabla^2 u = 0\) describes steady-state heat distribution. A "hot point" in the interior would cool down — contradicting steady state!

Consequences of the maximum principle: Uniqueness and continuous dependence on data for the Laplace BVP follow from the maximum principle. For the heat equation, these are established using an energy argument.

Existence follows from our ability to obtain solution formulas via separation of variables. For higher-dimensional problems with more complicated domains, solution formulas may not be available and existence must be established using specialized techniques.

ODE vs PDE: In ODEs, there is a very powerful existence and uniqueness theorem (Picard–Lindelöf) that applies in a wide variety of contexts. Unfortunately, no such universal theory exists for PDE — different techniques are needed depending on the equation and context.
